## üöÄ Getting Started

Follow these steps to run IDURAR locally:

1. **[Clone the repository](INSTALLATION-INSTRUCTIONS.md#step-1-clone-the-repository)**
2. **[Create MongoDB Atlas Cluster](INSTALLATION-INSTRUCTIONS.md#step-2-create-your-mongodb-account-and-database-cluster)**
3. **[Edit `.env` file](INSTALLATION-INSTRUCTIONS.md#step-3-edit-the-environment-file)**
4. **[Update MongoDB URI](INSTALLATION-INSTRUCTIONS.md#step-4-update-mongodb-uri)**
5. **[Install Backend Dependencies](INSTALLATION-INSTRUCTIONS.md#step-5-install-backend-dependencies)**
6. **[Run Setup Script](INSTALLATION-INSTRUCTIONS.md#step-6-run-setup-script)**
7. **[Start Backend Server](INSTALLATION-INSTRUCTIONS.md#step-7-run-the-backend-server)**
8. **[Install Frontend Dependencies](INSTALLATION-INSTRUCTIONS.md#step-8-install-frontend-dependencies)**
9. **[Start Frontend Server](INSTALLATION-INSTRUCTIONS.md#step-9-run-the-frontend-server)**

> Full guide: [INSTALLATION-INSTRUCTIONS.md](INSTALLATION-INSTRUCTIONS.md)
